# x3.组合逻辑运算

可以由基本逻辑运算组合成一些常见的逻辑运算。

# 1.与非

说明：两个输入，均为1时，输出为0，否则为1。

来源：先与再非

# 2.或非

说明：两个输入，均为0时，输出为1，否则为0。

来源：先或再非

# 3.异或

说明：两个输入，相同为0，不同为1。

来源：!AB+A!B

# 4.同或

说明：两个输入，相同为1，不同为0。

来源：AB+!A!B
