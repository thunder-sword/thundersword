# x2.记法

  • X为集合，x为元素
  • ∈和（不属于）
  • {x~1~,x~2~,x~3~,……}
  • X\Y（X减Y）表示一个集合，由在X中但不在Y中的所有元素组成的集合，是X的子集。
  • f:X→Y，表示“f是一个从集合X到集合Y的函数”
  • x:=y表示一个算术赋值操作，表示“将y赋予x”
  • 符号$△_=$表示“定义为相等”
